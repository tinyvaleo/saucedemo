## Описание
При входе под учётной записью `problem_user` вместо соответствующих изображений товаров в карточках отображается фото собаки.

## Шаги воспроизведения
1. Открыть страницу логина https://www.saucedemo.com/
2. Ввести `problem_user` в поле Username
3. Ввести `secret_sauce` в поле Password
4. Нажать кнопку Login
5. Проверить отображение карточек товаров на странице

## Ожидаемый результат
У каждого товара отображается соответствующее изображение.

## Фактический результат
Изображения отображаются некорректно (фото собаки).
